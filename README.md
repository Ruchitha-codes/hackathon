- Secure user authentication with email duplication check and security-themed background  
- User dashboard displaying profile details and options to upload text, images, or files  
- Advanced NLP detecting manipulation tactics, urgency, suspicious links, and abnormal patterns  
- Image and metadata analysis to uncover hidden threats
